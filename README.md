# Phonebook Management System

A console-based Phonebook Management System developed using the C programming language. This project allows users to manage contact information efficiently through a menu-driven interface.

## Features

- Add new contacts
- View all contacts
- Search contacts
- Update contact information
- Delete contacts
- File handling for data storage
- User-friendly menu interface

## Technologies Used

- C Programming Language
- File Handling
- Structures
- Functions
- Arrays

## Learning Outcomes

This project demonstrates practical implementation of:
- Data structures using structures
- File management in C
- Modular programming with functions
- CRUD (Create, Read, Update, Delete) operations
